Tags

Tags are arbitrary grouping mechanisms. The intent is that they're public, but this need not be the case. Each tag has:
  • A name
  • A description

and tags can be associated with programme items and people.

For example, you might want tags for:

  • Movies
  • Books
  • Comics
  • Blake's 7
  • Life on Mars
  • Fandom
  • Chaos
  • Children's programme
  • Science stream
and you might have tags for:
  • Committee
  • Tech crew
  • guests
  • Stewards
  • Green Room
  • Eclectic-21
  • ZZ9

It doesn't really matter - it's up to you. The point is, the database, by design, allows you to, say, "list all items that Fiona Scarlett is associated with", or "All items that Liz Williams is on", but not "All the items that have any guests on them"...

....well, okay, actually, it could, but there isn't a page for that, yet, so...

...so you can create tags for whatever you like, assign them to programme items and/or people, and then find all the items or people that are associated with that tag. That's it, but used properly, it could be very powerful (and very handy to members).

Enough pontificating.

Creating tags

You can create tags either:
  1. By going to the "Tags" page from the banner, then following "Create"
  2. By following "Tags: Create" on the Main page

When you do, you're prompted to enter the tag's name and description (the latter's optional). Then you click on "Add", and your tag is created.

Being an old page, clicking on Add just tells you that the tag has been created, and to go back to the main page.

Listing tags

You can list all the tags either:
  1. By going to the "Tags" page from the banner, then following "List"
  2. By following "Tags: List" on the Main page

Tags are listed in alphabetical order, with descriptions.

Each entry in this list is a link that'll take you to the tag's page, where:

  • You can Edit the tag (see below)
  • Any programme items associated with the tag are listed
  • Any people associated with the tag are listed

Editing tags

Whenever a tag appears, you can click on the tag (it's a link), and you'll go to the tag's page. There, you'll find a link to "Edit this tag". Following that link gives you a page where you can update the tag's name or description. Click on "Change" to save your changes.

Removing tags from things

If you want to remove a tag from a programme item or a person, click on the link of the respective thing; that'll take you to the programme item or person page, where all associated tags are listed. Click on the "Remove" link next to the tag to remove the tag from the programme item or person.

Deleting tags

You can delete a tag (so that the tag no longer exists):
  1. By going to the "Tags" page from the banner, then following "Delete"
  2. By following "Tags: Delete" on the Main page

Both give you a form where you can select the tag you want to delete. You have to tick the "Really delete" tick-box too, then click on "Delete".

When a tag is deleted, it's first removed from any programme items or people.
