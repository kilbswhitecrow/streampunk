Tags

Tags are arbitrary grouping mechanisms. Each tag has:
  • A name
  • A description
  • A flag, indicating whether it is visible in the public programme
  • An optional icon URL

and tags can be associated with programme items and people.

For example, you might want tags for:

  • Movies
  • Books
  • Comics
  • Blake's 7
  • Life on Mars
  • Fandom
  • Chaos
  • Children's programme
  • Science stream
and you might have tags for:
  • Committee
  • Tech crew
  • guests
  • Stewards
  • Green Room
  • Eclectic-21
  • ZZ9

It doesn't really matter - it's up to you. The point is, the database, by design, allows you to, say, "list all items that Buffy Summers is associated with", or "All items that Xander Harris is on", but not "All the items that have any guests on them"...

....well, okay, actually, it could, but there isn't a page for that, yet, so...

...so you can create tags for whatever you like, assign them to programme items and/or people, and then find all the items or people that are associated with that tag. That's it, but used properly, it could be very powerful (and very handy to members).

Enough pontificating.

Creating tags

You create tags by going to the "Tags" page from the banner, then following "New tag"

When you do, you're prompted to enter the tag's name and description (the latter's optional). Then you click on "Add", and your tag is created.

If you want the tag to have an ion as well, enter the URL of the icon. Note: icons should be small, so that they don't consume a lot of real estate on the screen.

Editing tags

Whenever a tag appears, you can click on the tag (it's a link), and you'll go to the tag's page. There, you'll find a link to "Edit tag". Following that link gives you a page where you can update the tag's name or description. Click on "Submit" to save your changes. {% endblock %}
